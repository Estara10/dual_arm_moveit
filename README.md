基于Franka双臂协作的物体搬运轨迹优化与柔顺控制研究
项目简介
本项目聚焦于仓储物流场景下长条形包裹的双臂协同搬运任务,基于Franka双臂协作机器人平台,探索双臂协作机器人的轨迹同步优化与柔顺力协调控制方法。
针对长条物体搬运中常见的姿态失稳、受力不均与轨迹不同步等问题,项目构建了"双臂同步抓取—协同搬运—精准放置"的完整任务流程,实现了轨迹优化算法与力位混合柔顺控制策略的协同设计。
技术特点

开发框架: ROS2 + MoveIt2
仿真环境: Gazebo
轨迹优化: CHOMP优化 + 等时参数化方法
控制策略: 主从协调 + 自适应阻抗控制
目标平台: Franka Panda双臂机器人系统
